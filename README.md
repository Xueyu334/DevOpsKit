# 🛠️ DevOpsKit (研发运维工具箱)

DevOpsKit 是一个基于 Vue 3 + Vite 构建的现代化、高性能的前端单页应用（SPA）。它不仅仅是一个单体工具，而是一个致力于提升研发与运维人员日常工作效率的 **聚合型工具箱导航门户 (Tool Hub)**。

目前平台的核心比对模块（Text Diff）已开发完毕，并在首页精心抽象并预留了诸如 JSON 工具、Base64 编解码、Hash 生成、URL 编解码以及 Cron 表达式生成等多种高可用工具的自动化扩展卡片入口。

## 🌟 核心特性与亮点

- 🎨 **现代化的极客视觉体验 (Premium UI/UX)**：
  - **灵动的起始页控制台**：采用类似原生操作系统应用启动台的卡片式网格布局。
  - **极其细腻的拟物交互**：深度定制的光影悬浮过渡、Icon 动态放大与倾斜变换、流光溢彩的炫彩文字渐变，以及多层次径向渐变背景，带来极致的新鲜感与 WOW 体验。
  - **零依赖的矢量视觉呈现**：摒弃沉重的外部图标库，全面采用了零依赖、行内加载的高清 Lucide 风格 SVG 图标矩阵。
  - **开箱即用的双态主题**：原生平滑适配全局高质感的浅色/深色（暗黑）双模式。
  - **敏捷的检索系统**：首页支持高度实时的关键字过滤，极速定位所需工具流。
- ⚡ **原生框架底座**：全面拥抱新一代基于 Composition API 的 **Vue 3** 及超高速的 **Vite** 前端构建工具。
- 🧩 **高优工程化配置**：
  - 全链路打通了 `unplugin-vue-components` 与 `unplugin-auto-import` 插件。涵盖绝大多数 Element Plus 组件和 Vue / VueUse API 的**全自动按需静默导入**，业务代码零 `import` 冗余。
  - 深度启用了打包时的 `manualChunks` 切片和大文件分包优化。
- 🔌 **高可扩展性数据驱动**：运用数据驱动视图的设计思想（Data-Driven Design），基于 JSON 列表配置化。后续每新增一个工具，只需在数组中增加一条元数据，首屏界面即可自动渲染出包含完整动效的工具卡片。

## 🔧 已集成与规划功能

### 1. 文本比对深度工具 (Text Diff Tool)
- 采用了高效的 `vue-diff` 核心比对计算引擎。
- 支持基于需求的“并排(Split)”与“统一(Unified)”双形态差异视图。
- 自带大容积长文本的 **虚拟滚动 (Virtual Scroll)** 防卡顿渲染能力，智能应对高达百万字长度的差异查勘。
- 提供纯文本、JSON、JavaScript、TypeScript、XML/HTML、CSS、Markdown 等跨语种精准语法高亮。
- 完美植入：**左右对调面板**、**一键极速清空**，并在双边内容完全一致时提供动态的绿色转场的成功状态气泡反馈。

### 2. JSON 深度解析工具 (JSON Tool)
- **双轨解析引擎**：同时提供严格标准 `JSON.parse` (Strict) 与高宽容度 `JS Eval` (Relaxed) 解析途径，完美应付含有未引键名、单引号或多余逗号的各种脏数据畸形 JSON。
- **极致性能 (Web Worker 加持)**：将深层抽象语法树的渲染与运算全面迁移至独立的 **浏览器后台工作线程 (Web Worker)**。就算面对上万节点的超大体积 JSON，主页面的 Vue 响应式数据绑定与输入体验依然丝滑无比、绝不锁死。
- **全方位的高质感视图**：深度定制开发了节点可视化 DOM 树。支持代码着色、数组索引透视、数据类型显式注标、并可任意展收无尽层级。
- **内置丰富指令集**：无缝内嵌了一键优雅格式化、极致单行压缩、字符串转义包裹以及部分反转义能力。

### 3. Base64 智能编解码工具 (Base64 Tool)
- **智能双向追踪引擎**：自动追踪用户最近操作、活跃输入的焦点区块，真正实现左右两侧面板**完全无差级的双向编码与解码**体验，任何一次计算输出都会智能、安全地投放至非活页对立面板，打破传统单向限制，再不阻扰覆写原有输入区数据。
- **全字符集与乱码强容错**：底层重构拥抱高标准原生规范 (TextEncoder API)，实现深层防腐级的 Unicode、中文、异构源等宽泛信息安全编码。针对非标数据具备前置的无效空格截获清洗能力。
- **独家多行批次引擎 (Multi-line)**：处理复杂长列表安全凭证或海量独立行文本时，一键解耦至独立行并发编解码：任何行的解析异常都不会影响其余数据的安全解析流转换。
- **贴心至极的微交互设计**：搭载基于纯前端无依赖原生剪切板 (Fallback 到 execCommand 底层兜底) 的无感**一键速拷 API**；在面板悬浮端提供基于超精准 Blob 数据缓冲流实时抓取的微米级毫秒 **字符统计及实际容量测算反馈 (Kb/Bytes 响应)**。

### 4. URL 编解码专家 (URL Encode Tool)
- **强转义模式动态感知 (encodeURIComponent / encodeURI)**：极度灵活的双重编码引擎，可在「保留安全 URL 结构」或「强制全参数转义拦截」之间自由取舍。完美防范网络 GET 请求、CORS 与鉴权携带时诸如 `?`，`&`，`=` 或是 `+` 的意外解析劫持。
- **全系特性复刻**：与 Base64 工具一脉相承的优质基因：毫无保留地集成了「**双边无损智能焦点焦点判断系统**」、「**高频全选一键复制交互 API**」、「**Blob 实时字节统计回显**」与「**批处理解耦式多行防护引擎**」。
- **前端特供强力纠错**：原生拦截含有残缺 `%` 或是非法 UTF-8 切片的 `URI Malformed` 并给与明确指示而有效制止主线程死锁。

## 📦 本地运行与部署指南

### 环境要求

请确保你的本地计算机或目标 CI/CD 容器中安装了 **Node.js** (推荐长效支持版 `v18+` 或更高版本)。

### 1. 拉取与环境装载

```bash
git clone https://github.com/Xueyu334/text-diff-tool.git
# 注: 未来仓库可能会迁移或更名为 devopskit
cd text-diff-tool
npm install
```

### 2. 研发环境热重载启动

```bash
npm run dev
# Vite 将光速开启本地研发服务器 (默认通常为 http://localhost:5173) 。
# 工程完美遵循移动端响应式规范，可支持通过局域网 IPv4 地址被同一局域网下的手机或平板终端直接调试访问。
```

### 3. 构建与生产部署

```bash
npm run build
```

打包流程默认将会自动闭环并销毁内部源码的 SourceMap（严格杜绝源代码层面的逻辑外泄并进一步减小发版包体积）。最终，高度压缩与防缓存哈希化后的纯静态态资源会统一被输出在项目根底下的 `dist/` 文件夹内。你可以直接将其上传至 Nginx，或丢入云原生静态应用托管平台（如 Vercel, Netlify, Github Pages 等）进行零复杂配置的 HTTP 访问挂载。
